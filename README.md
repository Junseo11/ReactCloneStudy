# Diary 클론 프로젝트

**공부 목표:**  
리액트를 공부하면서 Diary 애플리케이션의 기능을 모방하여 클론 프로젝트를 만드는 것이 목표입니다.
js 복습, 리엑트 공부진행

## 주요 기능

- **일기 작성**: 사용자가 새로운 일기를 작성하고 저장할 수 있는 기능
- **일기 목록**: 저장된 일기를 목록으로 표시하고 확인할 수 있는 기능
- **일기 수정 및 삭제**: 기존의 일기를 수정하거나 삭제할 수 있는 기능
- **일기 상세 보기**: 각 일기의 내용을 자세히 확인할 수 있는 기능
